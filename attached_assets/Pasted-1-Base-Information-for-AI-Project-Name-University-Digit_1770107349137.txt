Color Palette for University Meal System

1️⃣ Primary Color
Color: Deep Blue / Indigo


Hex: #1E40AF


Use For:


Top navbar / sidebar background


Buttons (primary actions: approve, submit)


Active menu items


Why:
Professional, trustworthy, educational vibe


Strong contrast with text



2️⃣ Secondary Color
Color: Soft Yellow / Amber


Hex: #FBBF24


Use For:


Highlights (e.g., pending orders)


Badges / notifications


Hover on cards or buttons


Why:
Evokes energy, attention, and appetite


Works well with deep blue



3️⃣ Background Colors
Main background: #F3F4F6 (light gray)


Card background: #FFFFFF (white)


Table row hover: #E5E7EB


Why:
Clean, readable, professional


Gives a modern dashboard look



4️⃣ Text Colors
Primary text: #111827 (dark gray/black)


Secondary text / captions: #6B7280 (medium gray)


Error / warning text: #DC2626 (red)


Why:
Clear readability


Standard professional UX



5️⃣ Accent Colors
Green (#16A34A) → approved orders, success messages


Red (#DC2626) → rejected orders, errors


Blue (#3B82F6) → info, neutral buttons


Amber (#FBBF24) → pending orders or notifications
